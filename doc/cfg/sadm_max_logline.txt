# sadmin.cfg SADM_MAX_LOGLINE Documentation
# Revise 2018_02_14
#----------------------------------------------------------------------------
The Log file name pattern is "[HOSTNAME]_sadm_[NAME_OF_SCRIPT].log".
Log are recorded in ${SADMIN}/log
There is a different log file per script, log are cumulative by default.
In python your can set 'st.log_append = False' or 'SADM_LOG_APPEND="N"' in 
shell script to create a new log when your script is started. 
SADM_MAX_LOGLINE represent the maximum number of lines that each *.log file
can contain. The log are trimmed to this number when the script is finished.
