# sadmin.cfg SADM_MAIL_TYPE Documentation
# Revise 2018_02_14
#----------------------------------------------------------------------------
Default Option for sending email after a script is ended
Can be overridden by changing SADM_MAIL_TYPE at the top of template script
[0] = Never send email      
[1] = Only send the log when script finish with error
[2] = Only send the log when script finish with success.
[3] = Always send the script log when script finish.
